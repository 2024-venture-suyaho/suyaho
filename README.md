## 📌 브랜치 활용

- **main 브랜치:** 최종 코드 관리 브랜치로, prototype 브랜치에서 문제가 없으면,merge 해서 사용

- **prototype 브랜치:** main 에 올리기전, 오류가 있는지 최종 점검하는 브랜치.
